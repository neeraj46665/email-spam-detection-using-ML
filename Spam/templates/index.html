Email Spam Detection



This online platform allows text analysis to identify potential spam messages. Enter text in the textarea, select an algorithm, and press CHECK for results.

Algo 1 (94% accuracy)

Random Forest Classifier

This scikit-learn pipeline uses TF-IDF to convert text into numerical features and a RandomForestClassifier to predict outcomes based on these features.

Algo 2 (95% accuracy)

Support Vector Classifier

This pipeline transforms text using TF-IDF and applies a Support Vector Classifier (SVC) with high penalty (C=1000) and automatic gamma setting for predictions.

Algo 3 (97% accuracy)

Multinomial Naive Bayes

This pipeline employs TF-IDF for text conversion and utilizes a Multinomial Naive Bayes classifier for making predictions based on the transformed features.

Algo 4 (97% accuracy)

Bagging Classifier

This scikit-learn pipeline uses TF-IDF to convert text into numerical features and a Bagging Classifier to predict outcomes based on these features.

Algo 5 (98% accuracy)

AdaBoost Classifier

This pipeline transforms text using TF-IDF and applies an AdaBoost Classifier with 100 estimators for predictions.
